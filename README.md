## Installation

Setup requires conda or miniforge.

a. Install conda or miniforge.

b. Clone this repository and open a terminal inside the repository folder:

    git clone https://github.com/charansamineni/brine_valorization.git
    cd brine_valorization

c. Create and activate the environment:

    conda env create -f environment.yml
    conda activate brine-valorization

d. Install IDAES solvers:

    idaes get-extensions
